# 营销数据分析工具 - 部署指南

## 方式一：Railway 部署（推荐）

### 为什么选 Railway？
- ✅ 免费 $5/月额度，初期够用
- ✅ 支持Docker一键部署
- ✅ 自带HTTPS域名
- ✅ 支持定时任务
- ✅ 自动从GitHub部署

### 步骤

#### 1. 上传代码到 GitHub

```bash
# 在本地解压代码包后
cd 营销数据分析工具

# 初始化Git
git init
git add .
git commit -m "Initial commit"

# 创建GitHub仓库后推送
git remote add origin https://github.com/你的用户名/营销数据分析工具.git
git branch -M main
git push -u origin main
```

#### 2. 在 Railway 部署

1. 访问 https://railway.app
2. 点击 "Start a New Project"
3. 选择 "Deploy from GitHub repo"
4. 授权GitHub，选择你的仓库
5. Railway会自动检测Dockerfile并部署

#### 3. 配置环境变量

在Railway项目设置中添加环境变量：

```
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=你的邮箱@gmail.com
SMTP_PASSWORD=你的邮箱应用密码
EMAIL_TO=接收报告的邮箱
```

#### 4. 访问应用

Railway会给你一个域名，如：
`https://营销数据分析工具-production.up.railway.app`

## 方式二：Render 部署（免费备选）

### 步骤

1. 访问 https://render.com
2. 创建 "Web Service"
3. 连接GitHub仓库
4. 配置：
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `uvicorn app.main:app --host 0.0.0.0 --port $PORT`
5. 添加环境变量（同上）

---

## 方式三：VPS 部署（适合有服务器的用户）

### 阿里云/腾讯云/华为云

```bash
# 1. 连接服务器
ssh root@你的服务器IP

# 2. 安装Docker
curl -fsSL https://get.docker.com | sh

# 3. 上传代码
scp 营销数据分析工具.tar.gz root@你的服务器IP:/root/

# 4. 解压并部署
cd /root
tar -xzvf 营销数据分析工具.tar.gz
docker-compose up -d

# 5. 配置Nginx反向代理（可选）
# 6. 配置域名和SSL证书
```

## 环境变量说明

| 变量名 | 说明 | 是否必填 |
|--------|------|----------|
| SMTP_HOST | SMTP服务器地址 | 可选（不发邮件可跳过） |
| SMTP_PORT | SMTP端口 | 可选 |
| SMTP_USER | 发件邮箱 | 可选 |
| SMTP_PASSWORD | 邮箱应用密码 | 可选 |
| EMAIL_TO | 接收报告的邮箱 | 可选 |

## 常见问题

### Q: 部署后访问不了？
A: 检查端口是否正确，Railway/Render会自动分配端口，不需要手动设置

### Q: 定时任务不执行？
A: 确保服务没有休眠（Railway免费版会休眠），建议升级付费或使用外部定时任务服务

### Q: 数据会丢失吗？
A: 使用了SQLite本地存储，容器重启数据会保留。建议定期备份
