System Information

AI features (DeepSeek) are configured by the server administrator. API Key is not exposed to users for security reasons.

AI Features

With DeepSeek API configured, you can enjoy the following AI-powered insights:

  • Dashboard Insights: Overall sales trend interpretation, top products analysis, and anomaly alerts
  • Diagnosis Insights: Sales diagnosis, conversion funnel analysis, and anomaly product explanation
  • Compare Insights: Multi-dataset comparison conclusions, ranking changes, and trend summaries
Without API Key, the system will provide basic rule-based insights (offline mode).
