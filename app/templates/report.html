Scheduled Report Settings
System automatically generates report at UTC 1:00 (Beijing 9:00) and sends to configured email.

To modify schedule, edit REPORT_CRON_HOUR and REPORT_CRON_MINUTE in config.py.

Email Configuration

Configure following environment variables to enable email:

Env VariableDescriptionExample
SMTP_HOSTSMTP Serversmtp.gmail.com
SMTP_PORTSMTP Port587
SMTP_USERSender Emailyour@email.com
SMTP_PASSWORDEmail Password/App Password******
EMAIL_TODefault Recipientdest@email.com
Security Tip: Do not write passwords in code. Use .env file or environment variables.
